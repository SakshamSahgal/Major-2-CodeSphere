
# To run Backend

`NOTE` : Make sure you have `Node 20` and `npm` installed

1. `cd Backend`
2. `npm install`
3. create a `.env`
4. save the required `.env` parameters
5. `node src/index.js`

# To run React Frontend

1. `cd Frontend`
2. `npm install`
3. create a `.env`
4. save the required `.env` parameters
5. npm start
